# Saint Paul - Hotel & Restaurant Website

This is the official repository for the **Saint Paul** Hotel & Restaurant web presentation.

## ✨ Features:
* **Modern Hero Section:** A sleek split-screen design focusing on luxury and visual identity.
* **Fully Responsive:** Completely optimized for mobile phones, tablets, and desktop devices.
* **Amenities Showcase:** Clean display of services (Free Wi-Fi, Parking, Breakfast) using FontAwesome icons.
* **Room Booking Grid:** A structured and intuitive grid layout showcasing accommodation units.

## 🛠 Technologies Used:
* **HTML5:** Semantic markup for better SEO and structure.
* **CSS3:** Custom layout using Flexbox and CSS Grid for precision and responsiveness.
* **FontAwesome 6.0:** High-quality vector icons for a professional look.
* **Google Fonts:** Premium typography featuring *Playfair Display* and *Poppins*.
